## 🚀 现场准备

来之前请准备好：

1. **笔记本电脑** — 建议带电源
2. **Node.js >= 22.12** — [下载安装](https://nodejs.org)
3. **API Key** — [免费获取](https://bailian.console.aliyun.com/cn-beijing/?source_channel=key_github&tab=app#/api-key)
4. **安装 CLI**：
   ```bash
   npm install -g bailian-cli
   bl auth login --api-key sk-xxxxx
   ```
