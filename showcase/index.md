# 用户案例 Showcase

来自社区成员的真实使用案例。每一个案例都是一次实践分享。

---

## 如何贡献你的案例

1. Fork [本仓库](https://github.com/ModelStudioAI/modelstudioai.github.io)
2. 在 `showcase/` 目录下新建一个 markdown 文件，命名格式：`YYYY-MM-DD-你的标题.md`
3. 按下方模板填写内容
4. 提交 PR

### 案例模板

```markdown
---
layout: doc
title: 你的案例标题
author: 你的 GitHub ID
date: YYYY-MM-DD
tags: [图文, CLI, 某行业]
---

## 我做了什么

简述你用 Model Studio AI 完成了什么任务。

## 使用的工具

- OpenWork / CLI
- 用了哪个 Skill
- 百炼模型版本（如适用）

## 步骤

1. ...
2. ...
3. ...

## 效果展示

（截图、产出文件、或简要描述效果）

## 踩坑记录

（可选：遇到了什么问题，怎么解决的）
```
